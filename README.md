# 🎓 Student Management System

A console-based Student Management System developed in Java using Object-Oriented Programming (OOP) concepts.

## 📌 Project Overview

This project allows users to manage student information through a simple console interface. It demonstrates the practical implementation of Java OOP principles and basic CRUD operations.

## ✨ Features

- Add a new student
- View all students
- Search for a student
- Update student information
- Delete a student
- Console-based menu system

## 🛠 Technologies Used

- Java
- Object-Oriented Programming (OOP)
- IntelliJ IDEA

- Polymorphism
- Classes and Objects

```bash
git clone https://github.com/your-username/student-management-system.git
```
